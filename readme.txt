== Description ==


Buffer Social widget offers 6 different social feed widgets that is Bloglovin, Facebook, Linkedin, Twitter, Instagram, and Pinterest. One can easily drag and drop the scoial feed widgets to add it availbe widget area.  


== Installation Process 1 ==


1. Go to WordPress Dashboard >> Plugins >> Add new >> Search Plugin 'Buffer Social Widget' >> Install >> Activate
2. Go to Dashboard >> widgets >> checkout all the 6 social widgets 
3. Drag and drop the widgets to show them at the desired location 
4. Save changes.  


== Frequently Asked Questions ==


= Can i make any changes to the way the Social Widget are Displayed =


We have used a default styles which will work with the themes further you can make changes as per needs.


= Is there way to insert the widgets using shortcodes =


No the plugin does not offer shortcodes option right now, Buffer social feed widgets can only be displayed in the widgets area avaible in the theme you are using.
